# Glossary / 用語集

本ドキュメントは、本ゲームにおける用語の定義を固定し、
設計・実装・バランス調整・議論における言葉の揺れを防ぐことを目的とする。

---

## ゲーム全体

### Game / ゲーム
本リポジトリで設計・実装されるボードゲーム。
19世紀（ヴィクトリア朝）を主軸に、21世紀時代への移行（Time Leap）を含む。

### Era / 時代
ゲーム内の歴史的フェーズ。
ルール、使用可能カード、移動手段、評価軸が変化する。

- Victorian：19世紀
- Modern：21世紀（現代）

---

## 盤面・地理

### Area / エリア
国家または地域を表す単位。
複数の Tile（マス）を内包する。

例：
- イギリス
- フランス
- インド
- 南アフリカ
- 北アメリカ(西海岸)
- 北アメリカ(東海岸）
- 南アメリカ
- 東ロシア
- 西ロシア
- 中国
- オーストラリア
- オスマントルコ

### Tile / マス
プレイヤーが移動・停止する最小単位。
マスには種類と効果が定義される。

### Route / ルート
マスまたはエリア間を接続する移動経路。
封鎖・開通・片道制限などの状態を持つ。

### Transport / 移動手段
ルートに紐づく移動方式。

- Ship：船
- Train：鉄道
- Land：陸路
- Special：特別移動（時代超越など）

---

## 行動・効果

### Action / 行動
プレイヤーが自分の意思で選択できる操作。
例：移動、トレード、鋳造、オークション参加。

### Effect / 効果
マス・カード・イベントによって発生する結果。
ゲーム状態を変更する最小単位。

例：
- 資産の増減
- 移動
- ルート開通
- 行動権限の付与

### Permission / 行動許可
一時的または恒久的に許可される行動制約の緩和・追加。
マスやカード効果として付与される。

例：
- 複数回トレード可能
- 特定通貨必須
- 対象プレイヤー数制限

---

## カード・イベント

### Event Card / イベントカード
引いた瞬間または条件達成時に効果を発動するカード。

### Deck / デッキ
イベントカードの山札。
Eraごとに構成が異なる。

### World Event / 世界イベント
全プレイヤー・全盤面に影響するイベント。
ルート開通や封鎖解除などを含む。

---

## 資産・経済

### Fungible Asset / 可換資産
数量で管理される資産。
- 金貨（グレード付き）
- 銀貨
- 銅貨
- 紙幣（Paper Money）
- 地金（Bullion）

### Non-Fungible Asset / 非可換資産
個体として管理される資産。

- 称号
- 鉱山権
- 工場
- 契約・保険

### Coin / 金貨
鋳造された個別資産。
ミント、時代、グレードを持つ。

---

## プレイヤー

### Player / プレイヤー
ゲームに参加する主体。

### Player State / プレイヤー状態
プレイヤーが保持する全情報。

- 現在地
- 資産
- 所持アイテム
- ステータス
- 一時効果

### Status / ステータス
数値または段階で表される評価・補正。

例：
- 称号ランク
- 信用
- 税率補正
- 没収リスク

---

## 評価・進行

### Scoring / 評価
勝利・解放条件に用いられる多軸評価。

- 身分
- 財源
- 金貨保有
- 鉱山・工業力
- 実績

### Milestone / マイルストーン
達成すると新要素が解放される条件。
例：時代移行、高位金貨鋳造、特別ルート開通。

---

## 特殊概念

### Time Leap / タイムリープ
特定条件を満たすことで時代を移行する仕組み。
称号や実績が必要。

### Lock / 封鎖
ルートや行動が使用不可の状態。

### Unlock / 開通
封鎖状態が解除され、使用可能になること。
